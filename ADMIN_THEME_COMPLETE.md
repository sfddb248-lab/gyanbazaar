## ğŸ”§ Customization

### Change Sidebar Color
Edit `assets/css/admin-ultra-theme.css`:
```css
:root {
    --sidebar-bg: #1a1d2e; /* Change this */
}
```

### Change Primary Gradient
```css
:root {
    --admin-gradient-1: linear-gradient(135deg, #YOUR_COLOR_1 0%, #YOUR_COLOR_2 100%);
}
```

### Adjust Animation Speed
```css
:root {
    --transition-base: 0.3s ease; /* Change duration */
}
```
