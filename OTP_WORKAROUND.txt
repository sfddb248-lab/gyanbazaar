═══════════════════════════════════════════════════════════
  HOW TO USE (DEVELOPMENT ONLY)
═══════════════════════════════════════════════════════════

STEP 1: User Signs Up
→ Go to signup page
→ Fill in details
→ Submit form
→ Redirected to OTP verification page

STEP 2: Admin Views OTP
→ Login to admin panel
→ Go to: Admin → View OTPs
→ Find the user's email
→ Copy the 6-digit OTP code

STEP 3: User Enters OTP
→ Paste OTP in verification page
→ Click "Verify OTP"
→ Account activated!
