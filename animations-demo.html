ðŸ“– Quick Usage Guide

Fade Animations:

  • .fade-in - Simple fade in effect
  • .fade-up - Fade in from bottom
  • .fade-down - Fade in from top
  • .fade-left - Fade in from right
  • .fade-right - Fade in from left

Animation Modifiers:

  • .delay-100 to .delay-500 - Add delays
  • .duration-fast - 0.3s duration
  • .duration-normal - 0.6s duration
  • .duration-slow - 1s duration

Glassmorphism:

  • .glass - Base glass effect
  • .glass-card - Glass card with hover
  • .glass-btn - Glass button
  • .glass-input - Glass input field
  • .glass-modal - Glass modal/overlay
  • .glass-badge - Glass badge

Color Variants:

  • .glass-primary - Blue tinted glass
  • .glass-success - Green tinted glass
  • .glass-warning - Yellow tinted glass
  • .glass-danger - Red tinted glass
